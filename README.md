# tic_tac_toe_tensorflow
tic_tac_toe_tensorflow is an implementation of tic tac toe game using Nural Q-learning in TensorFlow.

# requierments
- TensorFlow(0.12.0+)

# how to play
## human vs human
```
python tic_tac_toe.py 1
```

# how to train
```
python tic_tac_toe.py 0
```
